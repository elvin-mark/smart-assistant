# Smart Assistant
Simple Smart Assitant using tiny DL models:

- [ChatBot Model](https://huggingface.co/TinyLlama/TinyLlama-1.1B-Chat-v1.0)
- [Text To Speech Model](https://huggingface.co/facebook/mms-tts-eng)
- [Speech Recognition Model](https://huggingface.co/openai/whisper-tiny.en)
# Run Me
```sh
python src/main.py
```
